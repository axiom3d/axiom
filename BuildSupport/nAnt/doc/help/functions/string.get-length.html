string::get-length

Returns the length of the specified string.

Usage

int string::get-length(s)

Parameters

Name Type Description
s string input string

Return Value

The string's length.

Examples

  • string::get-length('foo') ==> 3
  • string::get-length('') ==> 0

Requirements
